## 시작 가이드
요구사항
- `npm 8.1.2`
- `node 16.13.2`
- `vue cli 5.0.8`
- `vue 3.5.6`

설치 방법
```
git clone www.naver.com  
cd Marine-vessel-imforamtion-App
```
```    
npm install
npm run serve
```
